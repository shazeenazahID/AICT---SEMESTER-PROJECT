# AICT---SEMESTER-PROJECT#
 The Portal Cafe - Frontend Project

## Project Overview

This repository contains the frontend website for *The Portal Cafe*, a static website created as the 1st semester project for the AICT (Introduction to Computing Technologies) course.

The website promotes a fictional campus coffee shop, highlighting fresh coffee, a cozy study-friendly environment, and affordable student prices.

- *Live Demo*: [https://shazeenazahid.github.io/AICT---SEMESTER-PROJECT/](https://shazeenazahid.github.io/AICT---SEMESTER-PROJECT/)
- *Technologies Used*: HTML, CSS
- *Project Type*: Static website hosted on GitHub Pages
